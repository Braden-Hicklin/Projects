    About

  • This Enemy Creation tool is intended to help speed up the process behind enemy creation for Dungeon Masters. The tool utilizes the Google HTML service API to submit data from the tool to a google sheet. The sheet is fitted with a custom extension script that is run every time the user submits an enemy via the create enemy button on the form. This tool can be used to help prepare massive amounts of enemies for potential encounters throughout a campaign in just minutes. To use the tool, simply click on the "Create Enemy" tab in the top right and fill out the sheet. Once the sheet has been filled out, press the button labeled "create enemy" at the bottom of the form, give it a second and a popup saying "Success form submitted" should pop up. From here you can simply navigate to your google sheet to confirm its submission. Have Fun!
